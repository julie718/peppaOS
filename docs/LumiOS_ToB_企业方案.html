LumiOS 不做定制工作流。Lumi 在日常使用中，自己感受、自己学习、自己长出来。

AI 人格引擎：不是工具，是存在

LumiOS 的核心不是对话模型，不是工作流引擎，是 AI 人格。

什么是 AI 人格

每个人用 AI 的方式都不一样。有人要它直接、有人要它温柔、有人要它先听再说。

Lumi 从交互中学习你的偏好——不是填问卷，不是选配置，是它在跟你聊天的过程中自己感受出来的。

情绪感知 + 波动模型

这是 LumiOS 真正的技术护城河：

  • 情绪感知：语速变快、用词变短、深夜还在打字——Lumi 知道你焦虑，调整回复节奏
  • 情绪波动模型：不是二分类"开心/不开心"。是一个连续的情绪状态空间，有强度、有衰减、有触发阈值
  • 人格一致性：无论情绪怎么波动，人格基底稳定——像人的性格一样，有情绪但不失自我

为什么企业需要这个

一个能感知情绪的管理者，比一个只会分派任务的管理者，团队绩效高 30%。

Lumi 对每个员工都一样——它记得你、感知你、用你最舒服的方式跟你工作。

部署方式与隐私保障

三个组件，一天部署

中心服务器 — Docker 一键部署在企业内网。负责组织记忆库的存储和检索、AI 人格的一致性同步、Token 配额的分发管理、以及管理仪表盘的 Web 服务。最低配置 4 核 8G，支持横向扩展。核心数据不出企业内网。

桌面客户端 — Windows + Linux 原生客户端（Tauri v2），装完即用。以桌面宠物形态低摩擦存在——不弹窗、不打扰、不强制。语音唤醒、对话、TTS 回复走完整链路。

企业 MCP 插件 — 通过 MCP 协议对接企业现有的办公工具。开箱即用的插件由灵序维护，企业也可以自行开发私有插件。

硬件延展：Lumi One 全息仓（会议室/高管办公室）· 智能台灯（开放式办公区光效交互）· 桌面机器人（前台接待）

接入流程：装客户端 → 组织码扫码加入 → 开始使用。一个下午全员上线。

管理仪表盘

中心服务器 Web 管理台，一眼看清组织 AI 运转状态：

  • 总览：多少人在用、每天用多少次、活跃曲线
  • 知识雷达：哪些部门沉淀了多少知识，哪里有缺口
  • 工作流图谱：Lumi 发现了哪些自动化机会，哪些已被采纳
  • 组织认知报告：每月自动产出 — 流程冲突在哪、知识孤岛在哪、效率瓶颈在哪
  • 情绪温度计：基于交互数据的团队情绪趋势——哪个部门在焦虑、什么时候士气低
不是监控工具。是组织智商 + 组织情商的仪表盘。
