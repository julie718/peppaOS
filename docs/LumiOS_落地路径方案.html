不花钱、不需要承诺、不改变现有工作模式。收入来自专业服务（项目制）：部署实施、MCP 定制开发、培训支持。

为什么我们现在能做这个

时机窗口

  • MCP 协议 2025 年底发布：Anthropic 推动的开放标准，现在恰逢标准确立期。先发优势就是在标准形成时已经在了
  • 企业 AI 落地失败率 > 90%：Gartner 数据，企业 AI 项目失败率居高不下——市场在主动寻找不一样的方案
  • PC OEM 急需差异化：联想/华为/Dell 的 AI PC 战略需要一个不跟 Copilot 正面冲突的互补品——Lumi 恰好是陪伴和人格
  • 开源 LLM 能力飞速提升：Qwen-3、DeepSeek-V3 等中文模型能力逼近 GPT-4o，本地推理成本趋近于零

我们的壁垒

  • 有一个能跑的产品——不是 PPT，是桌面端已可用的完整软件
  • 六个模块全通——桌面 + 语音 + 记忆 + 人格 + 情绪 + MCP，每个都是深度集成，不是 API 拼接
  • 核心护城河不可复制——大模型能力趋同，但人格数据不会。用户的每一次对话、每一个情绪波动，都在让 Lumi 变得独一无二且不可迁移
  • 开源社区已有基础——AGPL-3.0 开源，GitHub 有自然流量和开发者认知
