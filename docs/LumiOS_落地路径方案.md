LumiOS 不是一个 PPT 项目。桌面客户端今天就能跑。

**已交付的能力**：

| 能力 | 详情 |
|------|------|
| 桌面客户端 | Windows Tauri + Linux，系统控制全部打通 |
| 语音全链路 | 唤醒 → STT(Deepgram) → LLM(5 家可选) → TTS(CosyVoice 25 音色) |
| AI 记忆 | 关键词检索 + 置信度衰减 + LLM 提取 + 可视化面板 |
| AI 人格 | 4 个 preset + 自定义编辑器 + 在线人格市场 + 情绪感知引擎 |
| 情绪模型 | 语音频谱分析 + 文本情感追踪 + 行为信号识别，连续情绪状态空间 |
| MCP 生态 | 27 个工具，崩溃自动恢复，GitHub 直接导入 |
| 情绪感知 | 语速/用词/行为多维度实时分析，情绪波动模型连续追踪 |

## 第4页 · ToB + ToC 双线路线

### ToB：企业自主形成

企业内网部署中心服务器（Docker，4 核 8G）+ 员工装桌面客户端 + MCP 对接钉钉/飞书/企业微信/OA/邮件。

不花钱、不需要承诺、不改变现有工作模式。收入来自专业服务（项目制）：部署实施、MCP 定制开发、培训支持。

| 优先级 | 模块 | 说明 |
|--------|------|------|
| P0 | 中心服务器 + 企业 MCP 插件 | Docker 部署、飞书优先 |
| P1 | 工作流提取引擎 + 管理仪表盘 | LLM 模式识别 + /admin |

**目标**：Q3 3 家试跑 → Q4 10 家 → 2027 Q2 50 家

### ToC：硬件合作 + 市场抽成

桌面端免费铺量 → 手机伴侣 MVP → OEM 预装 → 硬件生态。收入来自自研硬件、OEM 分成、MCP 技能 + 人格市场抽成。

| 优先级 | 模块 | 说明 |

| P0 | OEM 安装方案 + 手机 App | 静默安装 + React Native 双端 |
| P1 | 跨设备同步 + 硬件 SDK | REST API + NPM 包 |

**目标**：Q3 5,000 → Q4 50,000 → 2027 Q2 500,000 用户

## 第6页 · 为什么我们现在能做这个

### 时机窗口

- **MCP 协议 2025 年底发布**：Anthropic 推动的开放标准，现在恰逢标准确立期。先发优势就是在标准形成时已经在了
- **企业 AI 落地失败率 > 90%**：Gartner 数据，企业 AI 项目失败率居高不下——市场在主动寻找不一样的方案
- **PC OEM 急需差异化**：联想/华为/Dell 的 AI PC 战略需要一个不跟 Copilot 正面冲突的互补品——Lumi 恰好是陪伴和人格
- **开源 LLM 能力飞速提升**：Qwen-3、DeepSeek-V3 等中文模型能力逼近 GPT-4o，本地推理成本趋近于零

### 我们的壁垒

- **有一个能跑的产品**——不是 PPT，是桌面端已可用的完整软件
- **六个模块全通**——桌面 + 语音 + 记忆 + 人格 + 情绪 + MCP，每个都是深度集成，不是 API 拼接
- **核心护城河不可复制**——大模型能力趋同，但人格数据不会。用户的每一次对话、每一个情绪波动，都在让 Lumi 变得独一无二且不可迁移
- **开源社区已有基础**——AGPL-3.0 开源，GitHub 有自然流量和开发者认知
